- pastikan php anda menggunakan versi php 8.1
- pastikan composer anda terupdate ke versi yang terbaru.
- di php harap tambahkan extension instalasi grpc (file ada di folder dll_tambahan).
- jangan lupa untuk mengaktifkan grpc di php.ini
. jika sudah di aktifkan restart xampp
- jika anda menggunakan xampp masukkan seluruh folder project ke folder xampp/htdocs
- jika sudah silahkan buka di terminal atau di vscode anda dan ketikan php artisan serve
- secara default project akan jalan di http://127.0.0.1:8000
